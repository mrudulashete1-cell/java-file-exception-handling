# ExceptionDemo (Java Exception Handling)

This is a simple Java program I created to understand how exception handling works.

## What this program does

* Performs division of two numbers
* Throws an error if division by zero is attempted
* Checks if a number is negative
* Uses a custom exception for invalid input

## Concepts covered

* try-catch block
* finally block
* throw keyword
* custom exception class
* handling multiple exceptions in a single catch

## How it works

* A divide method is used to perform division and throws an exception if the denominator is zero
* A custom exception is created to handle negative numbers
* In the main method, both cases are tested inside a try block
* Exceptions are caught and handled properly
* finally block runs at the end no matter what

## Sample Output

Exception caught: Cannot divide by zero
Finally block executed.

## Why I made this

I made this program to practice basic exception handling in Java and understand how different types of exceptions can be created and handled.
